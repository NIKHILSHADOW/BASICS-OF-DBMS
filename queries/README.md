# BASIC QUERIES

### Table

* We use CRUD operations 
* C - Create
* R - DESC
* U - Update
* D - Drop


### Row of a table

* C - insert
* R - select
* U - update
* D - delete

### Classification of sql commands

* DDL
* DML
* DCL
* TCL

* DDL(Data Defination Language)
    -   CREATE
    -   ALTER
    -   DROP
    -   TRUNCATE

* DML(Data Manipulation Language)
    -   Insert
    -   Delete
    -   Select
    -   Update

* DCL(Data Control Language)
    -   Grant
    -   Revoke

* TCL(Transaction Control Language)
    -   Rollback
    -   Commit
    -   Begin
